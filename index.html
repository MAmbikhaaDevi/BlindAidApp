How to Run This Application

This is a Next.js application. It is not a simple website that you can run by opening this HTML file directly in your browser. It needs a development server to run.

Instructions

  1. Open a terminal in your code editor (like VS Code).
  2. If you haven't already, install the necessary packages by running:
    npm install
  3. Start the development server by running the command:
    npm run dev
  4. Once the server is running, open your web browser and go to this address:
    http://localhost:9002

Following these steps will correctly start and run the full application.
